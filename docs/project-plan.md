# 나를 돌아보는 유언장 프로젝트 - 기획서

## 프로젝트 개요
사용자가 자신의 삶을 돌아보고, 소중한 사람들에게 전하고 싶은 메시지를 기록하는 웹 애플리케이션

## 핵심 가치
- **성찰**: 자신의 삶을 돌아보는 시간 제공
- **기록**: 소중한 순간과 감정을 보존
- **전달**: 사랑하는 사람들에게 마음을 전달

## 주요 기능

### 1. 메인 페이지
- 프로젝트 소개
- 시작하기 버튼
- 간단한 가이드

### 2. 작성 페이지
- **나의 이야기**: 자신의 삶을 돌아보는 섹션
  - 소중했던 순간들
  - 후회와 배움
  - 감사한 사람들
  
- **전하고 싶은 말**: 특정 사람에게 보내는 메시지
  - 수신자 이름
  - 관계
  - 메시지 내용
  
- **나의 가치관**: 삶의 철학과 가치관 기록
  - 중요하게 생각하는 것들
  - 삶의 교훈

### 3. 보관함
- 작성한 내용 목록
- 수정/삭제 기능
- 카테고리별 분류

### 4. 설정
- 프라이버시 설정
- 데이터 백업/내보내기

## 기술 스택 제안
- **Frontend**: React + TypeScript
- **Styling**: Tailwind CSS
- **Storage**: LocalStorage (초기) → 추후 Backend 연동
- **상태관리**: React Context 또는 Zustand

## 개발 단계
1. Phase 1: 기본 UI 및 로컬 저장
2. Phase 2: 사용자 인증 및 클라우드 저장
3. Phase 3: 고급 기능 (예약 전송, 미디어 첨부 등)
