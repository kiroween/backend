## 모바일 반응형 고려사항

- 768px 이하: 단일 컬럼 레이아웃
- 네비게이션: 햄버거 메뉴로 전환
- 카드 형태의 콘텐츠는 전체 너비 사용
- 터치 친화적인 버튼 크기 (최소 44x44px)

## 색상 팔레트 제안

- Primary: 차분한 블루/그린 계열 (#4A5568, #2D3748)
- Secondary: 따뜻한 베이지/크림 (#F7FAFC, #EDF2F7)
- Accent: 부드러운 골드/오렌지 (#ED8936, #DD6B20)
- Text: 다크 그레이 (#1A202C)
- Background: 화이트/라이트 그레이 (#FFFFFF, #F7FAFC)

## 타이포그래피

- 제목: 24-32px, Bold
- 부제목: 18-20px, Semi-bold
- 본문: 16px, Regular
- 캡션: 14px, Regular
- 한글 폰트: Noto Sans KR, Pretendard 추천
